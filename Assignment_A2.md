Assignment #2:

In assignment one I started developing the code with the idea of mimicking a heartbeat. I used the trigonometric functions to create pulses that were just slightly off from eachother, visualizing the pulsing nature of a heart. then I added a third pulse to create an arythmia in the hearts pulses. I tried to accent that interjecting nature by forcing that pulse to be a vertical gradiant and try and cut in on the horizontal pulses. I wanted to make the visual more chaotic as it was very rigid. I then added more trigonometric functions to push around the gradients and change how the colors changed over time. This created a very fluid affect that started to feel more spacey than like a heartbeat, but was still almost life-like.

I felt that leaning into the space like motifs was a good idea, so I added a circle in the middle. I included feedback with that wich created an almost speed like aspect to it. I wanted to keep playing with how trigonometric functions and see what ways I could affect the path of this circle. I also wanted different paths, this drove me to add additional circles with a different set of functions for a unique path. I then changed where I used the seconds() function and the frame variable to add different speeds, giving different characteristics to the motion of the gradient and spheres. The resulting visual felt reminiscent of Interstellar's blackhole simulation. I wanted to continue to explore that idea and stumbled onto making one of the spheres have a pulsing ring. After playing with the feedback, I found my final result. I felt that it really created an almost high speed space like effect that I'm really happy with.
